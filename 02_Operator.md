## 연산자 종류
### 대입
    a = 1
    
### 산술
    + 덧셈        2 + 5 = 7  
    - 뺄셈        2 - 5 = -3 
    * 곱셈        2 * 4 = 8
    ** 제곱       2**2 = 4
    / 나눈 몫     5 / 2 = 2
    % 나눈 나머지  5 % 2 = 1

```ruby
puts "더하기 : a+5 = #{a+5}"
puts "빼기  :  a-5 = #{a-5}"
puts "곱하기 : a*5 = #{a*5}"
puts "제곱 :  a**2 = #{a**2}"
puts "나머지 : a%3 = #{a%3}"
puts "나누기 : a/3 = #{a/3}"
```
